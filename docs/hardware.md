# Hardware Guide

## Raspberry Pi 5 Rig

Your order is well-matched to this project:

- **Pi 5 16GB** — headroom for INDI/Ekos, Starwatch API, plate solving, and light inference
- **1TB NVMe 2230** — fast boot, local image storage, optional Jellyfin for TV slideshow
- **GeeekPi metal case** — thermal stability for continuous or outdoor-adjacent use

## Telescope Connection

| Item | Recommendation |
|------|----------------|
| USB cable | Short (6–12"), USB-A to USB-B or USB-C adapter as needed |
| Placement | Pi inside/near case; cable protected from snow |
| Power | Scope internal battery or 12V astro power station |
| WiFi | Pi joins home router; agents connect over LAN |

## Winter / Outdoor Tips (NH)

- Enclose metal case in a ventilated weatherproof box with desiccant for long-term outdoor mounting
- No long USB runs — inches only
- Metal case provides heat dissipation; active cooler handles summer too
- Park scope and shut down Pi gracefully before storms

## Cables to Have on Hand

- USB-A to Micro-USB or USB-B (verify hand controller port)
- USB-C PD 27W power supply (usually in kit)
- Short ethernet cable (fallback if WiFi is weak in backyard)
- 12V barrel adapter if sharing scope power supply
