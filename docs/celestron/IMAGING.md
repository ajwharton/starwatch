# Imaging — 6D Mark II on Evolution 9.25

From SCT manual + Evolution manual (DSLR slew limits).

## Camera stack (your orders)

```
SCT rear cell
  → Celestron f/6.3 Focal Reducer/Corrector (#94163)
  → Celestron T-Adapter (#93633)
  → Lightdow Canon EF T-ring
  → Canon 6D Mark II
```

**Remove star diagonal** for photography — camera goes on rear cell, not through diagonal (SCT collimation guide).

## Slew limits (important)

Evolution manual: *“Using oversized accessories like DSLRs and T-adapters may require limiting the altitude angle to prevent striking the mount.”*

After alignment, in hand control or SkyPortal:

- **Altitude slew limit → Maximum** — reduce from 90° until camera clears fork (often ~70–80° while testing).

## Focus workflow

1. Align scope (SkyAlign).
2. Install camera stack (no diagonal).
3. Slew to **Moon** — live view 10×, focus.
4. Deep sky: reducer on, 30–45 s ISO 1600–3200, stack in Siril.

## Visual vs imaging

| Mode | Rear cell |
|------|-----------|
| Visual | Diagonal + eyepiece |
| Imaging | Camera stack only — cap visual back when swapping |
