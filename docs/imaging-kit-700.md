# $700 all-in imaging kit — Evolution 9.25

Complete shopping list for **deep-sky + planetary** on the NexStar Evolution 9.25, including **f/6.3 focal reducer** and adapters.

**Strategy:** **Keep Sony NEX-6 for planetary.** Buy one Canon for deep-sky. When used **6D Mark II** and **80D** cost the same → **get the 6D Mark II** (full-frame wins for DSO on f/10 SCT).

---

## Recommended: Canon 6D Mark II kit (~$690–730)

### When 6D II ≈ 80D in price → 6D II every time

| | 6D Mark II | 80D |
|--|------------|-----|
| Sensor | **Full-frame** 26 MP | APS-C 24 MP |
| Deep sky at f/6.3 | **Wider field, cleaner noise** | Good but tighter, noisier |
| Planetary video | 1080p60 | 1080p60 |
| **Your plan** | **DSO body** | Skip — NEX-6 does planets anyway |

Full-frame matters on a 9.25" SCT: even with the reducer, APS-C is a noticeably tighter crop on M31/M42.

### Stack order (rear of scope → camera)

```
9.25" SCT rear cell
  → f/6.3 Focal Reducer/Corrector
  → SCT T-Adapter
  → Canon EF T-ring
  → Camera body
```

**Remove the focal reducer** for rare Canon lunar shots. **Planets → NEX-6** at f/10.

## Used-market tips

- **6D Mark II:** Shutter &lt; 80k OK; confirm WiFi + articulating screen work
- **Battery:** 6D II officially uses **LP-E6N** (LP-E6 fits in practice)
- **Reducer:** Buy new if no return policy; used from trusted astro seller is fine
- **Skip:** original **6D** if Mark II is same price — Mark II adds 1080p60, better AF, 26 MP FF
